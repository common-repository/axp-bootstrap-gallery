== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'AXP Bootstrap Gallery'
3. Activate BuddyPress from your Plugins page. (You will be greeted with a Welcome page.)

= From WordPress.org =

1. Download AXP Bootstrap Gallery.
2. Upload the 'AXP Bootstrap Gallery' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate BuddyPress from your Plugins page. (You will be greeted with a Welcome page.)
